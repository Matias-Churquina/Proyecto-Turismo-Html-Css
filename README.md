# 🌍 Proyecto Turismo - HTML y CSS

Una plataforma web responsiva de turismo desarrollada con **HTML5**, **CSS3** y **JavaScript**, utilizando **Bootstrap 5** para crear una experiencia visual atractiva y moderna.

## 📋 Descripción del Proyecto

Este proyecto es una página web completa para una agencia de turismo que permite a los usuarios explorar diversos destinos turísticos alrededor del mundo. La plataforma ofrece información sobre paquetes de viaje, precios, agencias asociadas, y un blog de viajes.

## ✨ Características Principales

- **Página de Inicio (Home)**: Hero section con video de fondo, destinos destacados, carrusel de testimonios y contador animado de visitantes
- **Catálogo de Destinos**: Grid de destinos con filtrado por categoría (Sol y Playa, Naturaleza, Cultural, Gastronómico, Religioso)
- **Modales Interactivos**: Detalles de cada destino con imágenes, descripciones y precios
- **Tabla de Precios**: Tarifas estimadas de los paquetes turísticos
- **Blog de Viajes**: Artículos y contenido sobre viajes
- **Página de Contacto**: Formulario para consultas
- **Agencias Asociadas**: Información de agencias de viaje
- **Banco de Macros**: Paquetes especiales predefinidos
- **Diseño Responsive**: Adaptable a dispositivos móviles, tablets y desktop
- **Animaciones Suaves**: Efectos visuales con jQuery para mejorar la experiencia del usuario

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos personalizados y diseño responsivo
- **JavaScript**: Interactividad y lógica del cliente
- **jQuery**: Simplificación de manipulación DOM y animaciones
- **Bootstrap 5.3.8**: Framework CSS para componentes y grid system
- **Font Awesome 6.5.2**: Iconografía
- **Google Fonts**: Tipografías personalizadas

## 🚀 Cómo Usar

### Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a internet (para CDN de Bootstrap y Font Awesome)

### Instalación

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/Matias-Churquina/Proyecto-Turismo-Html-Css.git
   ```

2. **Navegar al directorio del proyecto**:
   ```bash
   cd Proyecto-Turismo-Html-Css
   ```

3. **Instalar dependencias** (opcional):
   ```bash
   npm install
   ```

4. **Abrir en el navegador**:
   - Abre el archivo `index.html` directamente en tu navegador
   - O usa un servidor local como Live Server en VS Code
